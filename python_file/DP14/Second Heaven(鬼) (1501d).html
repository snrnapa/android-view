解説

  • 捻りが混ざる16分のリズムがかなりの難易度で、それ以外にも8分のリズムで埋め尽くされていて忙しい。 -- 名無しさん (2010-01-31 23:35:47)
  • 最大の難所は2回目の「Somebody's Scream!」の直前の連続16分3連地帯。中央を軸に16分で捻りが入るため、接続するにはスライドが無難。交互でやるには強靭な腰の力が必要。 -- 名無しさん (2011-05-26 10:28:11)
  • ↑スライドでなくても、パネルの中央に体の軸を残したまま1P↑2P↑へ足を伸ばすように交互に踏めば繋がると思います。 -- 名無しさん (2011-05-27 15:58:17)
  • 典型的な局所難であり、転調直前の１６分３連４セットのカズダンスは当時としては足１６以上でもあまり見ない超振り回し配置。序盤から１６分３連階段渡りもあり、ＳＰの１６分捻り滝とは違った方向で難しい。スコア難易度は完全に足１５の領域で、恋閃鬼に勝るとも劣らない -- 名無しさん (2013-12-05 21:29:57)
