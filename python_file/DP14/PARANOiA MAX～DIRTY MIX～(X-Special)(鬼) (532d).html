  • あまり話題にならないが、かなりの遠距離に捻りと着物踊を強化したような譜面で詐称寄り。交差渡りとスイッチこそＸスペにしては少ないが、振り回しは強めで左右のみの渡りから強制縦連など嫌らしい要素が多め。目玉の超遠距離は４分間隔なので反復横飛びで同時スイッチして両端を拾うか、全て↓経由なので内側か片側どちらか固定して踏むかのどちらか。ＥＳＰで言う６３連地帯からが本番で、最初の滝は１Ｐ→を左足の遠距離ビジで取る左向き固定がやりやすいかも。その後の滝は着物踊やエタラブ鬼に似た遠距離振り回しだが、↑の遠距離繰り返しなので↑を目押しして右向き固定と覚えておく。 -- 名無しさん (2013-11-21 22:25:23)
  • ↑遠距離の繰り返しは右向き固定じゃなくて内側振り回しの間違いだった、失礼。その箇所はサナモレ激の開幕を繋げた配置なので、あらかじめサナモレ激と着物踊をやっておくと良さげ -- 名無しさん (2013-11-22 22:30:54)
