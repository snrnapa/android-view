解説

  • ホームポジション同時スタート。ESPより更に密度が減り16分は3連地団駄止まりで数も控えめに。代わりに厳しめの振り回しと遠距離ビジの8分滝が局所的にありSPとは差別化されている。とは言え難しい配置は僅かであり、ノーツもかなり少ないので局所難の足13強でも納得出来る範疇。問題の遠距離ビジ滝は↓ビジ遠距離が2回の↑ビジ遠距離が1回で、後者は後ろ足で取る必要があり踏み外しやすい。どうしても踏めない場合はスライド箇所を決めて左向き固定で踏むと回避出来るので一考の余地に。 -- 名無しさん (2020-02-15 01:58:19)
