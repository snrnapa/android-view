  • BPM推移：160-151-155-152-146-152-136-119-110-(停止)-110-180
  • 踊をノーツ増やして強化した譜面。RAINBOWが有効。クリア基準である難易度は妥当か強。フルコンに関してはLv不相応の12分入りの縦ジャンプがある。この早さは270の８分相当。停止や徐々に低速になる序盤のギミックから考えて足14ではかなり難しい。 -- 名無しさん (2013-12-26 02:08:04)
  • ＥＳＰが足１５弱とするなら、こちらも足１５弱と思われる詐称枠。遠距離１２分２連スキップや捻り始動地団駄１２分５連など、ＤＰならではの踏み難い配置も…長い空白後の滝は４分段階から上ビジ姿勢で左足で拾うと、直後の１２分滝が踏みやすくなる。正面から入る場合は後交差地団駄になる為、正面入りしたいならミラーも一考の余地有り -- 名無しさん (2013-12-26 21:52:56)
