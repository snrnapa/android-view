解説

  • この譜面の特徴はなんと言っても16分の後ずさり渡り。速度が尋常ではなくその部分の難易度は高い。 -- 名無しさん (2010-01-31 23:50:25)
  • 数回のスキップ地帯があるが、全て非常に素直で難易度は高くない。 -- 名無しさん (2010-01-31 23:50:43)
  • スキップと8分だけならともかく、16分3連の渡りやラストの16分5連渡りはBPM340の8分相当で、局所的にはMAX系を上回る。 -- 名無しさん (2010-09-06 00:28:01)
  • Ｘ３にて１３→１４に昇格。スキップ箇所はスイッチすると交差渡り箇所が踏みやすくなるので、要使い分け。１６分３連渡りは局所的に見ると足１６クラスの速度。後退りの正規より前歩きのミラーの方が楽なのでどうしても間に合わない時はミラー推奨。ラストの１６分５連渡りもミラーの方が踏みやすい -- 名無しさん (2013-12-02 22:24:35)
