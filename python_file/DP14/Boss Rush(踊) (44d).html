解説

  • BPM推移 : 190-95-190
  • BPM95地帯は同時2連+16分滝という構成が多いのだが、捻らせまくる上に、交互踏みを阻止するかのような意地悪な配置も混じっている(111combo〜)。 -- 名無しさん (2018-05-19 08:04:03)
  • 後半は2p↑ 1p→ 1p↑のように、大股で一気に渡らせる配置が複数ある。捻り振り回しで疲弊したところに降ってくるので、身体がついて行けず事故りやすい。 -- 名無しさん (2018-05-19 08:19:52)
  • ホームポジションスタート。低速含めて同時からの入り足が非常に迷いやすく、全交互には相当暗記が必要。交互に踏めたとしても相当振り回す配置で１６分も局所で自重していないので、移植曲踊譜面としては最高難易度で明らかな詐称。 -- 名無しさん (2018-06-10 19:05:45)
