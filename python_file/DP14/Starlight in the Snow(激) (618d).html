解説

  • 正統派足14譜面でありESPのような鳥フェニ配置こそないものの、16分3連が絡んだ交差渡りと振り回しで難易度を上げている。
  • 後半FA地帯手前にある16分3連最後の316コンボ目からの「2P↑・1P→・1P↑」の8分3連が唐突な遠配置になっているため、接続面で要注意。
  • 終盤の400コンボ目前後にキモプリ配置が2回来るため、交互で取るにせよスライドで取るにせよ意識が必要。
  • ホームポジションスタート。ESPより16分周りが簡単になり、同足始動の16分3連ビジや16分滝が無くなり5連止まりになっている。代わりにアフロ捻りと交差を含めた遠距離配置が目立ち、8分はかなり振り回される配置となってる。前交差のみだが交差から16分3連を踏んだり終盤の遠距離ビジから交差する配置は難解で、交互に踏めない内は苦戦するが全体的には足14弱で留まる範囲。 -- 名無しさん (2020-04-07 17:21:14)
