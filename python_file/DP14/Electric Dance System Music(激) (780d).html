解説

  • BPM推移 : 140-(停止)-140-(停止)-140
  • DDR AのシステムBGMをプレイアブル曲にしたもの
  • 開幕から前後交差捻りを踏まされ、アフロ捻り１６分２連はＤＰにおいては規格外の配置で前代未聞。二度目の停止手前の６パネ８分滝は極めて交互に踏み辛いので、チュンリーで対処するのも手。その停止もＦＡに隠れた１６分２連になっており、明けを空打ちで合わせ辛いのも嫌らしい。以降も交差８分滝に１６分５連が混ざった配置が来るが、交互に踏めないと悉く１６分の入り足を間違えるので踏めないと悲惨。スタトレより強く、間違いなく詐称だろう -- 名無しさん (2016-06-14 00:30:08)
  • 「DANCE MUSIC」手前の箇所は、赤矢印の左足始動（青矢印は右足）と覚えておくといい。後は矢印の通りに足を動かすだけ（それが異常に難しい）なのだが、途中の1P↓の青矢印周辺の配置が後ろ交差が絡んでいるためコンボを切りやすい。 -- 名無しさん (2017-07-02 01:32:52)
  • 地味に難所なのが、終盤の16分5連地団駄が絡んだ交差渡り。何が問題かと言うと、前半と後半で16分5連地団駄の始動する足が違うこと。最初の16分5連地団駄2回は「左足始動」で続く青矢印を「右足」で踏む、後半の16分5連地団駄2回は「右足始動」で続く青矢印を「左足」で踏む。 -- 名無しさん (2017-07-02 01:38:17)
  • A20にて足13→足14に昇格。
