解説

  • BPM推移 : 150-75-150-75-150-300
  • VOLTAGEとCHAOSが100を超えていることから分かる通り、高密度低速箇所が難関。75BPM地帯で普通に32分どころか24分が混じるため、体感的には足15～16譜面の低速箇所を踏んでいるかのよう。当然単押しと同時押しの見極めも大事。
  • 最後は300BPMの8分カニ配置でフィニッシュだが、逆に言えば「300BPMまでしか加速しない」ということなので、HSはx1.75～2.0、見えるならばx2.25あたりに設定した方がいい。当然最後を暗記するという手もあるが、足14ということもあり、低速を見切る力を養った方が早い。
  • 幸い全体的な配置自体は足14相応。EXTRA STAGEの突破を考えるだけならば、基本BPMが150ということもありスライドやチュンリーでごまかす手段も一応は有効。
    • 余談だが、DPにおいてSA鬼譜面を除き、一つ下の難易度とのLv差が7もある珍しい譜面。SPではΔMAXの足5BSP→足13DSPのLv差8が最大。
  • 低速は密度が高いが音合せなので予習するとかなり楽になる。ただし微妙に配置が遠い所があるので踏み外さないように注意。 -- 名無しさん (2019-05-20 01:18:47)
  • BPM150地帯は地団駄と軽い振り回しだが一度だけ16分9連。 -- 名無しさん (2019-05-20 01:20:12)
  • BPM300地帯は4分2連→8分3連の繰り返しから8分3連×2 -- 名無しさん (2019-05-20 01:23:16)
